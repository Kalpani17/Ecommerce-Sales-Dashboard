# E-Commerce Sales Analytics Dashboard

This project is an interactive E-Commerce Sales Analytics Dashboard built using SQL Server and Power BI. The dashboard analyzes sales, orders, customers, products, revenue, payment methods, order status, customer behavior, and product performance.

## Tools Used

- SQL Server
- SQL Server Management Studio
- Power BI Desktop
- DAX
- Git & GitHub

## Dashboard Pages

1. Executive Overview
2. Sales Analysis
3. Customer Analysis
4. Product Analysis

## Key Features

- Total revenue analysis
- Total orders and completed orders
- Customer analysis by city and gender
- Product performance analysis
- Revenue by category, brand, and payment method
- Monthly revenue trend
- Interactive slicers for filtering

## Database Tables

- Customers
- Products
- Orders
- Order_Details
